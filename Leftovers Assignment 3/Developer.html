Developer

The developer needs to have a good understanding of programming and the applications they use to develop the project. Communication and team-work skills are needed to accurately create the project as specified by the other team members. They need to be able to tell if a function can be implemented or not, and if the task either takes too long or is not feasible with the available resources. The developer also should have some management skills to properly sort out how the project can be done within the team's skills and goals.

Skills and Experience

  • Excellent coding background
  • Minimum 1-year programming experience
  • Experienced in one or more of the following coding languages
  • JavaScript
  • HTML
  • Smartphone Development
  • Python
  • CSS
  • PHP
